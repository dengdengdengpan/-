# 网易云音乐-移动端网页版

### 技术栈

- HTML：语义化的HTML
- CSS：使用flex布局，CSS 动画实现复杂效果
- JavaScript：原生JavaScript实现交互
- localStorage：本地存储，记录用户的搜索记录
- LeanCloud：后端提供歌曲数据

### 实现功能

- 首页
  - 推荐歌单，最新音乐
- 热歌榜和歌单页面
- 播放页面
  - 歌曲播放暂停
  - 歌词滚动
  - 唱片旋转效果
- 搜索页面
  - 输入单曲、歌手、专辑可进行搜索
  - 搜索记录本地存储
